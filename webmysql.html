WebMySQL

Overview

A web interface to a MySQL database server.

Features

  • Supports the following SQL queries:
    • SELECT
    • DESCRIBE
    • JOIN
    • SHOW
  • Advanced query options such as LIMIT and DISTINCT
  • Fully customisable look and feel
  • HTML results or downloadable CSV file

Requirements

  • Linux operating system
  • Perl installed
  • Perl MySQL modules
  • MySQL server
  • Web server with CGI support
